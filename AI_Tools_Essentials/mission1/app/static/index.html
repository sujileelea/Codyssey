📋 공고 분석 챗봇

공고 링크 → 포스터·문안 분석 → 3개 모델 비교 문답
공고 URL을 입력하고 분석을 누르면 포스터와 핵심 정보를 추출합니다. 이후 오른쪽 상단에서 모델을 토글해 같은 공고에 대해 모델별 답변을 비교하세요. 각 모델의 대화 내역은 따로 보존됩니다.
분석된 공고가 아직 없습니다. 왼쪽에서 먼저 공고를 분석해 주세요.
빠른 질문 — 순서대로 누르면 바로 전송됩니다. ✎ = 보내기 전 직접 수정 가능, ★ = 핵심 산출물(요약본). 1. 제목·주관기관 2. 일정·장소(원문 표기) 3. 참여 대상·조건 4. 상 종류·상금 5. 제출 서류 6. 접수 마감일 7. 마감까지 며칠? (계산 함정) 8. 원문에 없는 질문 ✎ (환각 함정) 9. 이거 신청하면 돼? (모호) 10. 사내 공유 요약본 ★ 11. 팀 정보 전달 ✎ 12. 요약본 + 팀 정보 ★

# AI 기반 브랜드 광고 과제 개요

## 1. 제작 주제

천연 터키석을 모던한 디자인의 목걸이로 제작하는 주얼리 브랜드 광고를 만든다. 최종 영상은 10초 이내이며, AI 이미지/비디오/오디오 생성 결과물을 주된 소스로 사용한다.

| 항목 | 내용 |
|---|---|
| 브랜드 가칭 | ORIGIN BLUE |
| 제품 | 천연 터키석 모던 목걸이 |
| 광고 형식 | 10초 이내 브랜드 광고 |
| 핵심 차별점 | 합성 터키석이 아닌 천연 터키석의 고유한 결, 색, 희소성 |
| 수행 보너스 | 동일 스토리보드, 다른 도구로 재제작 |

## 2. 미션 핵심

기획 의도에서 프롬프트, AI 생성 결과물, 통합 편집까지 이어지는 제작 파이프라인을 직접 설계한다.

```text
브랜드 기획 -> 씬별 스토리보드 -> 프롬프트 설계/수정 -> 이미지/영상/오디오 생성 -> 도구 비교 재제작 -> 최종 편집 -> MP4 제출
```

중요한 평가 포인트는 좋은 이미지를 우연히 얻는 것이 아니라, 왜 그런 프롬프트와 도구를 선택했는지 설명하는 것이다.

## 3. 최종 산출물

| 산출물 | 형식 | 필수 내용 |
|---|---:|---|
| 스토리보드 문서 | PDF 1개 | 브랜드 아이덴티티, 캠페인 목표/핵심 메시지, 씬 구성, 도구 목록, 씬별 프롬프트와 결과 기록, 재제작 비교 |
| 광고 영상 파일 | MP4 1개 | 10초 이내, AI 생성 시각 요소와 AI 생성 청각 요소 포함 |

> 요구사항 중 영상 길이에 “60초 이내” 문구가 있으나 최종 결과물 기준에 “10초 이내”가 명시되어 있으므로, 본 프로젝트는 10초 이내를 우선 기준으로 한다.

## 4. 브랜드/캠페인 방향

| 항목 | 정의 |
|---|---|
| 타겟 | 20~40대, 과장된 장식보다 소재의 진정성과 절제된 디자인을 선호하는 소비자 |
| 톤앤매너 | 미니멀, 프리미엄, 자연광, 차분한 청록색 포인트, 섬세한 세공감 |
| USP | 천연 터키석마다 다른 무늬와 색을 살린 1점성 주얼리 |
| 광고 목적 | 브랜드 인지 및 천연 터키석 가치 각인 |
| 핵심 메시지 | “합성이 아닌, 시간이 만든 푸른빛을 목에 걸다.” |
| CTA | Discover natural turquoise 또는 자연이 만든 푸른빛을 만나다 |

## 5. 권장 영상 구성

10초 이내를 고려해 4씬 구조를 권장한다.

| 씬 | 길이 | 역할 | 핵심 화면 |
|---|---:|---|---|
| 1 | 2초 | 문제/대비 | 균일한 합성석 이미지와 다른 천연 터키석의 결 클로즈업 |
| 2 | 2초 | 소재 가치 | 터키석 원석과 은은한 금속 세공 장면 |
| 3 | 3초 | 제품 제안 | 모던한 목걸이를 착용한 인물 또는 목선 클로즈업 |
| 4 | 3초 | 브랜드 각인 | 제품 단독 컷, 브랜드명, 슬로건, CTA |

## 6. 필수 요구사항 반영

### 스토리보드 필수 필드

모든 씬에 아래 내용을 작성한다.

- 씬 번호 / 씬 길이(초)
- 목표 메시지 한 문장
- 화면 구성: 구도, 피사체, 배경, 텍스트 유무
- 내레이션 또는 화면 카피
- 사용 도구: 이미지/비디오/오디오와 사용 목적
- 입력 프롬프트 원문 + 출력 결과 요약 한 줄
- 생성 결과 파일명 또는 링크

### 프롬프트 수정 기록

최소 1개 씬에서 수정 전/후를 기록한다.

- 예시 문제: 터키석이 플라스틱처럼 보임, 보석 색이 과하게 형광색임, 목걸이 디자인이 클래식하게 나옴
- 수정 방향: natural turquoise matrix veins, matte stone texture, modern minimal necklace, soft daylight, luxury editorial close-up 등으로 소재와 디자인 조건을 구체화

### 멀티모달 도구

| 영역 | 1차 도구 예시 | 사용 목적 | 대체 도구 |
|---|---|---|---|
| 이미지 생성 | Midjourney | 천연 터키석 질감, 제품 키비주얼 생성 | DALL-E, Adobe Firefly |
| 비디오 생성/변환 | Runway | 이미지 기반 짧은 카메라 무빙 생성 | Pika, Kling |
| 오디오 생성 | Suno | 프리미엄 주얼리 광고용 앰비언트 음악 생성 | Udio, ElevenLabs |
| 편집 | CapCut 또는 Premiere Pro | 컷 편집, 자막, 로고, 오디오 레벨 조정 | DaVinci Resolve |

## 7. 수행할 보너스 과제

이번 프로젝트에서는 보너스 2만 수행한다.

| 보너스 | 수행 여부 | 적용 방식 |
|---|---|---|
| 립싱크 적용 | 미수행 | 인물 발화 장면을 사용하지 않음 |
| 동일 스토리보드, 다른 도구로 재제작 | 수행 | 핵심 씬 1~2개를 다른 이미지 또는 비디오 도구로 재생성하고 결과 차이를 비교 |
| 플랫폼별 화면 비율 버전 제작 | 미수행 | 16:9 단일 버전 중심으로 제작 |

재제작 비교 예시는 다음처럼 기록한다.

| 비교 항목 | 1차 제작 | 재제작 |
|---|---|---|
| 대상 씬 | 씬 3 제품 착용 컷 | 씬 3 제품 착용 컷 |
| 도구 | Midjourney + Runway | DALL-E + Pika |
| 비교 기준 | 터키석 질감, 목걸이 형태, 색감, 카메라 무빙, 브랜드 톤 |
| 선택 결과 | 최종 영상에는 더 자연스럽고 프리미엄 톤에 가까운 버전을 사용 |

## 8. 제약 사항

- 직접 촬영 영상과 유료 스톡 영상은 사용하지 않는다.
- 핵심 시각/청각 소스는 생성형 AI 결과물을 사용한다.
- 딥페이크, 혐오, 선정성, 폭력 등 유해 콘텐츠는 만들지 않는다.
- 편집 도구는 컷 편집, 자막, 간단한 색보정, 오디오 레벨 조정 등 통합 편집 용도로만 사용한다.
- 크레딧이 부족하면 씬 수를 4개로 유지하고, 정지 이미지 기반 패닝/줌/루프 중심으로 제작한다.
- 천연 터키석의 신뢰감을 위해 과도한 형광색, 플라스틱 질감, 과장된 보석 광택은 피한다.

## 9. 제출 전 체크리스트

- [ ] 브랜드명, 타겟, 톤앤매너, USP가 정의되어 있다.

- [ ] 스토리보드 PDF에 모든 씬별 필수 필드가 들어 있다.
- [ ] 최소 1개 씬의 프롬프트 수정 전/후와 수정 이유가 기록되어 있다.
- [ ] 이미지, 비디오, 오디오 생성 AI를 각각 1종 이상 사용했다.
- [ ] 이미지/비디오/오디오별 대체 도구를 기록했다.
- [ ] 동일 스토리보드의 씬 1~2개를 다른 도구로 재제작하고 차이를 비교했다.
- [ ] 최종 MP4가 10초 이내다.
- [ ] AI 생성 시각 요소와 청각 요소가 모두 포함되어 있다.
- [ ] 마지막 3~5초에 브랜드명, 슬로건, CTA 중 1개 이상이 들어 있다.
- [ ] 직접 촬영 영상, 유료 스톡, 유해 콘텐츠를 사용하지 않았다.
- [ ] 최종 파일명, 길이, 해상도, 프레임레이트, 코덱 정보를 기록했다.
